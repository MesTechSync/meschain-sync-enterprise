## 🚀 Quick Start

### 🎨 Frontend Development (Cursor Team)
```bash
cd frontend
npm install
npm run dev
npm run build
npm run test
```

### 🤖 Backend Setup (VSCode Team)
```bash
cd backend
composer install
php migrate.php
php artisan serve
./vendor/bin/phpunit
```

### 🚀 DevOps Deployment (MUSTI Team)
```bash
cd deployment
./production-deploy.sh
./monitoring-setup.sh
./backup-configure.sh
```
